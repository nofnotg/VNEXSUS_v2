# 법무 / 동의 / 안전장치 플레이스홀더

## 목적
이 문서는 실서비스 오픈 전 필요한 법무/동의 문서의 뼈대를 먼저 확보하기 위한 템플릿 문서다.
현재 버전은 구조 확정용이며, 공개 오픈 전 반드시 정식 문안으로 정밀화해야 한다.

## 1. 필수 문서 목록
1. 서비스 이용약관
2. 개인정보 처리방침
3. 민감정보(건강정보) 수집·이용 별도 동의
4. 개인정보 처리위탁 고지
5. 국외 이전 안내/동의 (해외 클라우드/API 사용 시)
6. AI 분석 결과 안내문
7. 전문가연결 동의문
8. 마케팅 수신 동의(선택)

## 2. 회원가입 체크박스 구조
### 공통 필수
- [필수] 서비스 이용약관 동의
- [필수] 개인정보 수집 및 이용 동의
- [필수] 민감정보(건강정보) 수집 및 이용 동의
- [필수] AI 기반 문서분석 결과가 참고용 안내임을 확인
- [필수] 개인정보 처리위탁/국외 이전 안내 확인(해당 시)

### 선택
- [선택] 마케팅 정보 수신 동의

### 일반사용자 결과 전
- [필수] 본 결과는 참고용 안내이며 확정 판단이 아님을 확인
- [선택] 전문가 상담 연결 동의

### 조사자 추가
- [필수] 의뢰자료 업로드 권한 보유 확인
- [필수] 업무 목적 이용 확인

## 3. 최소 문구 축
### 서비스 이용약관
- 서비스 목적
- 계정 책임
- 금지행위
- 결제/환불 기본 구조
- 서비스 변경/중단
- 책임 제한
- 분쟁 해결

### 개인정보 처리방침
- 수집 항목
- 수집 목적
- 보유 기간
- 제3자 제공 여부
- 처리위탁
- 파기 절차
- 권리행사
- 문의처
- 변경 고지

### 민감정보 동의
- 건강정보를 처리한다는 사실
- 처리 목적
- 보유 기간
- 동의 거부 권리와 불이익

### AI 안내문
- 자동화 분석 한계
- 결과는 참고용
- 최종 판단은 개별 사안 검토 필요

### 전문가연결 동의
- 연결 요청 정보 전달 범위
- 보유 기간
- 철회 방법

## 4. 구현 요구사항
- 동의 버전 관리
- 동의 시각 저장
- IP / userAgent 저장
- 약관 버전 변경 시 재동의 가능 구조
- 문서 페이지 라우팅 분리
- 관리자에서 현재 활성 버전 관리 가능

## 5. 공개 오픈 전 반드시 정밀화
- 실제 수집 항목
- 실제 위탁사/수탁사
- 실제 국외 이전 여부
- 보유/파기 기간
- 환불 정책
- 연락처/담당자
- 아동 관련 정책
