# Historical Phase Docs Archive

## 목적

이 폴더는 Phase 01 ~ Phase 12.6 문서를 원본 훼손 없이 별도 위치에서 보기 쉽게 정리한 아카이브다.

## 원칙

- 원본 phase 문서는 루트 위치에 그대로 둔다.
- 이 아카이브는 탐색 편의를 위한 복제본이다.
- 기술 사실은 바꾸지 않는다.
- 원본이 여전히 authoritative source 이다.

## 범위

- `phase01*` ~ `phase12_6*` 루트 문서
- `phase12_5_runtime`
- `phase12_5_compare`
- `phase12_6_runtime`
- `phase12_6_compare`

## 폴더 규칙

- phase prefix 별로 하위 폴더를 나눈다.
- 예:
  - `phase01`
  - `phase02_5`
  - `phase12_6`

## 주의

- 이 폴더는 product or phase13 planning 문서를 담지 않는다.
- 여기 있는 파일을 수정하더라도 원본 frozen artifact 가 수정된 것이 아니므로, 실제 기준 문서는 항상 루트 원본을 확인해야 한다.
